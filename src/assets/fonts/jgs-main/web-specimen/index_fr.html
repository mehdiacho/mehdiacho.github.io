EN English version
================================================================================
     
Jgs Font est une famille de fontes faite en hommage à Joan G. Stark (aka jgs,
Spunk), pionnère de l'art ASCII.

Les glyphs qui composent Jgs Font peuvent se combiner d'un caractère sur
l'autre, d'une ligne sur l'autre. Ainsi à partir de caractères uniques il est
possible de produire des lignes continues, des trames et des motifs.

Dans un soucis de pouvoir changer de taille de corps tout en conservant ces
effets de continuité au pixel près, la famille est declinée en trois fontes.

Jgs5 pour les corps multiples de 10 : 10px, 20px, 30px etc.
Jgs7 pour les corps multiples de 14 : 14px, 28px, 42px etc.
Jgs9 pour les corps multiples de 18 : 18px, 36px, 54px etc.


Notes
-----

Jgs Font un projet encore inachevé. Certains caractères sont encore manquants
et les correspondances formelles entre les corps restent à travailler.

Vous trouverez ici une table des caractères et de comparaison entre les corps.
